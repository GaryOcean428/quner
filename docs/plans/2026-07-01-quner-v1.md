# quner v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `quner` — a portable, capability-detecting Linux efficiency daemon that tunes CPU governor / RAPL / NVIDIA power-cap to the EXP-132 work-per-joule interior optimum, installs as a root systemd service+timer, and is validated in a zero-risk local sandbox before any real write.

**Architecture:** Standalone package (zero qig-applied import). Ports the proven primitives from `qig-applied/src/qig_applied/efficiency/{telemetry,daemon}.py` and `inference/accelerate.py`, adds a `QUNER_SYSFS_ROOT`/`QUNER_NVIDIA_SMI`/`QUNER_STATE_DIR` override layer (safe-by-construction), a RAPL PL1 writer, snapshot/rollback, workload profiles, memguard, reverse-PRIME, systemd install, and a CLI. EXP-132 is an internal calibration table (the only physics tie).

**Tech Stack:** Python ≥3.10, stdlib-only core (optional `nvidia-ml-py` extra), `uv` build, `pytest`, systemd, docker+bwrap for sandbox.

## Global Constraints

- Python ≥ 3.10; stdlib-only core (optional extra: `nvidia-ml-py`). Copied verbatim from spec §13.
- **Zero import of `qig_applied` / any QIG package** — vendor primitives (spec §4).
- **Every sysfs/proc path** built from `os.environ.get("QUNER_SYSFS_ROOT", "/sys")` / `"/proc"`; `nvidia-smi` from `QUNER_NVIDIA_SMI`; state dir from `os.environ.get("QUNER_STATE_DIR", "/var/lib/quner")` (spec §6).
- **fail-loud**: absent lever → explicit "unavailable", never silent success (spec §2, §6).
- **Every mutating command** supports `--dry-run` that writes nothing (spec §6, §12).
- Governor writes keep the **double-allowlist** (`_KNOWN_GOVERNORS` ∩ live `available_governors()`) (spec §4).
- Objective default is `work_per_joule` (interior optimum), not `work_per_s` (spec §7).
- reverse-PRIME **never on install**, dry-run-first, reboot-gated (spec §9).
- Commit after every green task.
